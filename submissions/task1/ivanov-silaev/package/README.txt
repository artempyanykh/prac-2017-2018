Задача поиска цены игры и оптимальных стратегий антагонистической матричной игры.

На данном этапе выполнены следующие пункты:
1. Реализована функция nash_equilibrium, которая по матрице, определяющей антагонистическую матричную игру, находит цену игры и оптимальные стратегии игроков.
2. Добавлены примеры различный игр, а так же организован красивый вывод результатов.
3. Решение оформлено в виде пакета.

Для успешного запуска необходимы следующие пакеты: numpy, scipy
